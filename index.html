Features

Built for people
who plan ahead.

Most budgeting apps show where your money went. Nexus-Fin shows where it's headed.

📈
10-Year Wealth Projection
Compound growth modeling based on income growth, expense inflation, and investment returns. Best/base/worst case sensitivity curves.
🎯
Goal Tracking
Set milestones — emergency fund, home down payment, retirement corpus — and see exactly when you'll hit them based on current behavior.
🔍
Spend Anomaly Detection
AI flags categories trending above your 3-month average. Catch lifestyle creep before it derails your plan.
💼
Multi-Asset Portfolio
Track Equity, Debt, Gold, and Real Estate in one view. XIRR-based returns per asset class with allocation heatmap vs target.
💰
SIP Tracker
Monthly SIP inputs auto-roll into your projection. See maturity value at any future date instantly.
📊
Savings Rate Benchmark
Calculates your savings rate and compares against the 50/30/20 rule — know exactly where you stand.

How It Works

From input
to insight.

01

Enter Your Numbers

Add income sources, recurring expenses by category, current investments, and financial goals. Everything is stored locally — nothing is sent anywhere.

02

Projection Engine Runs

Monthly surplus (income − expenses) is calculated, then compounded forward at your chosen investment return rate. Income growth and inflation are applied year over year.

03

Goal Gap Analysis

For each goal, the engine compares your projected portfolio value at the target date against the target amount — showing exactly how much more (or less) you need.

04

Visual Dashboard

Net worth curve, category-wise expense waterfall, and goal progress rings render via Chart.js — updating live as you adjust any input.

05

Optional AI Insights

If enabled, anonymized spend category data is sent to Claude API for anomaly detection and personalized suggestions — never raw transaction details.

Setup

Running it
takes seconds.

Step 1
Clone or download
terminal bash 
$ git clone https://github.com/MAHAYAVANSHI2904/Nexus-Fin.git
$ cd Nexus-Fin
Step 2
Open in browser
No install, no server. Drag index.html into Chrome or Firefox — or double-click it.
terminal bash 
$ open index.html   # macOS
$ start index.html  # Windows
Step 3 (optional)
Enable AI spend analysis
Get a free key from console.anthropic.com and paste it into the in-app Settings panel. Stored only in your browser's localStorage — never sent to any server but Anthropic's.

Privacy & Security

No server.
No account.

🔒

Fully Client-Side

All financial data — income, expenses, investments, goals — lives only in your browser's localStorage. Nothing is uploaded.

🙈

No Tracking

No analytics, no cookies, no third-party trackers. The app doesn't know who you are.

🧠

AI Sees Categories, Not Transactions

If AI analysis is enabled, only anonymized category totals are sent — never line-item transactions or account details.

🗑️

One-Click Reset

Settings → Reset wipes all stored data instantly. No account to delete, no data to request back.
